A Memory Game

Thank you for visiting this website. In order to celebrate everything BMW we have created a fun mini game that will test your memory skills. This website and its contents is best enjoyed on a desktop.

You may find it easier to have numbers shown rather than the logo. This will be useful for mobile users. If so please click the numbers button.

When you are ready.

Step 1 - Press the start button. This will hide the unrelated images.
Step 2 - Click on a tile and then another.
Step 3 - Remember the tile pictures (they reset after 1 second) and try to match all of them as quickly as you can.
Please note: if you click on the tiles too quickly it "may" flag as an incorrect score.
If you want to start again please press the reset button.
