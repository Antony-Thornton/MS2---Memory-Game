A Memory Game

Thank you for visitng this website. In order to celebrate everything BMW we have created a fun mini game that will test your memory skills. This game is best enjoyed on a desktop.

When you are ready.

Step 1 - Press the start button. This will hide the header image.
Step 2 - Click on a tile and then another.
Step 3 - Remember the tile pictures and try to match all of them as quickly as you can.
Step 4 - If you want to log your score please use the submit.
If you want to start again please press the reset button.
